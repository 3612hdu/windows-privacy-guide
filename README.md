# Windows 11 隐私深度解析（2026 版）

一个关于 Windows 11 隐私机制的系列文章。每章聚焦一个主题，说清楚它是什么、怎么工作、你能做什么。

**不卖产品。不做焦虑营销。不给你的隐私"打分"。** 只提供你在 Windows 设置 App 和官方文档中找不到的技术细节。

## 为什么要读这个

- Windows 设置 App 告诉你"遥测有基本/增强/完整三档"，但**不说每档具体包含什么**
- 微软隐私声明是公开的，但分散在几百页文档里
- 大部分"Windows 优化指南"只告诉你怎么关，不告诉你关了会怎样

这系列文章填补这个信息差。
