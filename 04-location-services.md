## 一、Windows 怎么确定你的位置？三种定位方式

Windows 11 使用三种方式确定你的位置，按精度从高到低排列：

### 1.1 GPS（精度 ~5 米）
如果你使用的是带 GPS 芯片的设备（Surface Pro、部分笔记本），Windows 直接读取卫星定位数据。

### 1.2 Wi-Fi 三角定位（精度 ~30 米）
Windows 扫描附近所有 Wi-Fi 热点的 MAC 地址和信号强度，将这些数据发送到 Microsoft 的位置服务数据库。数据库返回一个估计位置。**这个数据库是微软自己维护的**——通过 Bing 地图采集车和用户设备被动上传建立。

关键点：**即使你没有连接任何 Wi-Fi，只要 Wi-Fi 开关是开的，Windows 就能通过扫描附近热点来定位。**

### 1.3 IP 地址定位（精度 ~城市级别）
最粗糙的方式。根据你当前的公网 IP 查询地理位置数据库。通常只能精确到城市。

## 二、位置数据的流动路径

当你使用需要位置的应用时（比如天气 App），完整的数据流是这样的：

```
应用请求位置
  → Windows 位置平台（检查应用是否有位置权限）
    → 如果用户允许且系统位置服务开启：
      → 优先尝试 GPS（如果有硬件）
      → 如果 GPS 不可用，扫描附近 Wi-Fi 热点
        → 将 Wi-Fi MAC 地址列表发送到 Microsoft 服务器
          → Microsoft 返回估计位置
      → 如果 Wi-Fi 不可用，使用 IP 地址查询
    → 将位置返回给应用
```

**你的 Wi-Fi MAC 地址列表（即你周围有哪些 Wi-Fi 热点）被发送到 Microsoft 服务器**——这是位置服务中最容易被忽略的隐私细节。

## 三、位置服务的注册表路径

系统级位置开关：
```
HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\CapabilityAccessManager\ConsentStore\location
String: Value = Deny
```

这个值设为 `Deny` 后，所有应用都无法获取位置。设为 `Allow` 则允许（默认）。

应用级别的权限存储在：
```
HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\CapabilityAccessManager\ConsentStore\location\NonPackaged
```
每个应用的权限状态以子键形式存储。

## 四、静默位置访问——应用在后台偷偷定位

Windows 11 允许应用在**后台**访问位置——即使你并没有在使用那个应用。根据 Windows 的隐私文档：

- 应用可以在后台**每 15 分钟**获取一次位置更新
- 如果应用声明了"后台位置"能力，甚至可以更频繁
- 任务栏右下角的位置图标会在任何应用访问位置时亮起——**如果你看到它在不该亮的时候亮了，说明有应用在静默获取你的位置**

你可以通过设置 → 隐私 → 位置 → 逐个禁用应用的"后台位置"权限来控制。

---

## 五、位置历史记录

Windows 会保留**24 小时的位置历史记录**（存储在本地设备上）。这些记录对以下应用可见：
- 任何已授权位置权限的应用
- Cortana/Windows 搜索（用于提供本地化的搜索结果）

定期清除位置历史对隐私有益。位置历史的存储路径：
```
%LocalAppData%\Microsoft\Windows\LocationCache
```

## 六、关闭位置服务的实际影响

如果你关闭整个位置服务：

| 受影响的功能 | 替代行为 |
|------------|---------|
| 天气 App | 使用你手动输入的城市（而非自动定位） |
| 地图 App | 无法导航到当前位置 |
| 查找我的设备 | 无法定位设备（但该功能在台式机上通常不准确） |
| 时间自动时区 | 不受影响（时区基于 IP，不需要精确位置） |
| 附近共享 | 不依赖位置服务 |
| 搜索结果 | 不再本地化，但搜索词仍可能暴露你的城市（如果包含地名） |

**对大多数台式机用户来说，关闭位置服务的实际影响非常小**——台式机通常没有 GPS 芯片，Wi-Fi 定位也不够精确，而 IP 只能定位到城市。

---

## 七、一个常见误解

很多人以为"关掉 Wi-Fi 就不会被定位了"。实际情况是：

- **台式机**：关掉 Wi-Fi 后只能通过 IP 定位（精度：城市级别）。如果你用有线网络，Windows 本来就只通过 IP 定位。
- **笔记本**：关掉 Wi-Fi 后，如果你没有 GPS 芯片，位置服务基本失效。但如果你在公共场所连接过 Wi-Fi，**Microsoft 的 Wi-Fi 位置数据库可能已经记录了你的热点位置和 MAC 地址**。

所以真正的隐私保护策略是：
1. 关闭位置服务（设置 → 隐私 → 位置 → 关）
2. 清除位置历史（设置 → 隐私 → 位置 → 清除）
3. 或者：保留位置服务，但逐个审查哪些应用有权访问（默认大部分应用没有权限）

---

## 八、底线

位置服务对你的实际价值取决于你的设备类型和使用场景：

- **带 GPS 的平板/笔记本 + 经常移动**：位置服务有实际用途（地图、天气、设备找回）
- **台式机 + 固定位置**：位置服务对你几乎没有价值。你的 IP 地址已经足够暴露你的城市。

后者的话，建议关闭。
