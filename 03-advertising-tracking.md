## 一、Windows 的广告系统是怎么工作的？

大多数用户以为"广告追踪"是浏览器的事——清掉 Cookie 就没事了。但在 Windows 上，有一层更底层的追踪机制：**操作系统级别的广告 ID**。

这个机制的工作原理：

1. Windows 在系统安装时生成一个**唯一的广告标识符**
2. 任何 UWP 应用都可以通过 API 读取这个 ID（不需要用户许可）
3. 广告网络用它**跨应用**关联你的行为：应用 A 看到你搜了耳机 → 应用 B 给你推耳机广告
4. 这个 ID 是**设备级**的，不是用户级的——同一台电脑上不同账户共享同一个广告 ID

## 二、广告 ID 的技术细节

### 2.1 生成机制

广告 ID 在以下时机生成：
- 系统首次启动时
- 用户手动点击"重置广告 ID"后

存储位置（注册表）：
```
HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\AdvertisingInfo
```

关键值：
- `Enabled` (DWORD): 1 = 允许使用广告 ID, 0 = 禁用
- `Id` (String): 实际的广告 ID 值（类似 `00000000-0000-0000-0000-000000000000` 格式的 GUID）

### 2.2 读取权限

**任何 UWP 应用都可以读取广告 ID**，只要它在应用清单中声明了 `internetClient` 能力——几乎所有需要联网的应用都会声明这个能力。不需要额外的用户授权弹窗。

Win32 桌面应用无法直接通过 API 读取广告 ID，但可以通过以下方式间接获取：
- 通过 `Windows.System.UserProfile.AdvertisingManager` API（需要引入 UWP 互操作）

### 2.3 重置 vs 关闭

Windows 提供两个不同操作：
- **关闭广告 ID**：应用读到的是空字符串。新应用无法追踪你，但已经收集了你旧 ID 的广告网络可能仍然保留历史数据。
- **重置广告 ID**：生成一个新 ID。相当于"换了一张脸"，但追踪机制仍然存在。

## 三、"量身定制的体验"——第二层追踪

除了广告 ID，Windows 还有一个叫"量身定制的体验"的选项。这是 Microsoft 自己的追踪系统，工作原理如下：

1. 收集你的诊断数据（包括完整级别的浏览历史）
2. 在 Windows 提示、通知、设置建议中推送**定制化内容**
3. 根据你的设备使用模式推荐 Microsoft 产品和服务

这个系统的独特之处：
- 它不是第三方广告，是 Microsoft 自己的推荐引擎
- 它使用的是**诊断数据管道**，不是广告 ID 管道
- 它默认开启
- 关闭它不会影响诊断数据级别——你仍然在发送诊断数据，只是不再用于个性化推荐

注册表路径：
```
HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Privacy
DWORD: TailoredExperiencesWithDiagnosticDataEnabled = 0
```

## 四、应用启动追踪——你打开什么应用，Microsoft 都知道

Windows 10/11 默认会追踪你**启动的每一个桌面应用**（不仅仅是 UWP 应用）：

注册表路径：
```
HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced
DWORD: Start_TrackProgs = 0
```

这个值设置为 1 时，开始菜单的"最常用应用"列表会基于你的启动频率排序。但数据也会被遥测系统收集（在"完整"级别下）。

---

## 五、跨设备追踪：你的手机和电脑被关联了

如果你用同一个 Microsoft 账户登录了多台设备，以下数据会跨设备关联：
- 广告 ID（不同设备有不同 ID，但通过账户关联）
- 浏览历史（通过 Edge 同步）
- 活动历史（应用/文件打开记录）
- 位置历史（如果开启了位置服务）

这意味着：**你在手机上搜了某样东西，Windows 电脑上就会看到相关广告。** 这不是巧合——这是 Microsoft 广告平台的标准功能。

关闭跨设备关联的唯一方法是：
1. 使用本地账户登录 Windows（不用 Microsoft 账户）
2. 或者在 privacy.microsoft.com 上管理跨设备数据共享

## 六、哪些数据被用于广告？（汇总表）

| 数据源 | 用于广告？ | 如何关闭 |
|--------|----------|---------|
| 广告 ID | ✅ 是（跨应用追踪） | 设置 → 隐私 → 常规 → 关闭"广告 ID" |
| 诊断数据 | ✅ 是（量身定制的体验） | 同上 → 关闭"量身定制的体验" |
| 浏览历史 | ✅ 是（完整级别遥测 + Edge 同步） | 调低遥测级别 + 关闭 Edge 同步 |
| 位置数据 | ✅ 是（本地化广告） | 设置 → 隐私 → 位置 → 关闭 |
| 应用启动记录 | ✅ 是（开始菜单推荐 + 遥测） | 注册表 Start_TrackProgs=0 |
| Cortana/搜索词 | ✅ 是（搜索广告） | 关闭云端搜索 |

## 七、总结：你对广告追踪的实际控制力

如果你**不做任何配置**（Windows 11 默认状态）：
- 广告 ID：开
- 量身定制的体验：开
- 遥测：完整(3)
- 应用追踪：开
- 你的设备在网络上的可识别度：极高

如果你**按本系列第一章的建议操作**：
- 广告 ID → 关
- 量身定制的体验 → 关
- 遥测 → 基本(1)
- 应用追踪 → 关
- 你的设备在网络上的可识别度：大幅降低（仍有 IP 级别的粗粒度追踪）

**关键的诚实结论：你无法在 Windows 上实现完全的"零追踪"。** 但你可以把追踪从"个性化广告+行为画像"降到"系统基本运行数据"，差距是质变的。
